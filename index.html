Research Project · Deep Learning

ABTSC

Automated Brain Tumor Segmentation & Classification

Brain tumor diagnosis from MRI scans is slow, manual and prone to error. ABTSC changes that — it takes an MRI scan as input, automatically segments the tumor region using a U-Net model, and classifies the tumor type using a CNN with 97.6% accuracy. Built with a Flask backend and React frontend, the system processes scans end-to-end and delivers results instantly. Trained on 3,264 labelled MRI images across four tumor categories.

🧠
U-Net deep learning model for precise tumor region segmentation from MRI scans
🎯
CNN classifier identifying Glioma, Meningioma, Pituitary and No Tumor types
⚙️
5-stage MRI preprocessing — noise reduction, skull stripping, CLAHE contrast enhancement
📡
Flask REST API backend handling image upload, model inference and result delivery
🖥️
React frontend for uploading MRI scans and displaying tumor type with confidence score
📊
Trained on Kaggle brain MRI dataset — 3,264 images with 80/20 train-test split

ShareCircle

Community Borrowing & Lending Web App

ShareCircle is a web application that helps people in a community share items they aren't using. Instead of buying something you only need once, you can borrow it from someone nearby. The app handles the entire flow — listing items, sending borrow requests, approvals, returns and notifications. Built with real-world security and usability in mind.

Firebase authentication with Firestore security rules
📦
Full borrow, approve, return and confirm flow
🔔

🛡️
